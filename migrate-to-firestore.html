🔧 HydroStorePro — Firestore Setup & Migration Tool

This is a standalone, separate tool. It does not touch your live HydroStorePro app or modify your browser's local data in any way — it only reads the local data and copies it to Firestore.

⚠️ Before using this: open firebase-config.js notes below, paste in your real Firebase project keys, and create an Authentication user (email/password) in the Firebase Console first — this tool needs to sign in to write to Firestore.
